# 🛠️ Requisitos Funcionais (RF)

Os requisitos funcionais descrevem as funcionalidades que o sistema deve oferecer para atender às necessidades dos usuários e do negócio.

---

| ID | Requisito | Descrição |
| :--- | :--- | :--- |
| **RF01** | Manter Filmes | O sistema deve permitir cadastrar, editar, excluir e consultar filmes (Título, Duração, Sinopse, Elenco, Diretor, Gênero). |
| **RF02** | Manter Unidades | O sistema deve permitir a gestão das unidades de cinema (Nome, Endereço, Capacidade de Público). |
| **RF03** | Gerenciar Sessões | O sistema deve permitir a criação e organização de sessões diárias por sala e cinema. |
| **RF04** | Registro de Público | O sistema deve permitir o registro diário do público presente em cada sessão realizada. |
| **RF05** | Consulta de Filmes | O sistema deve permitir a busca de informações sobre elenco, diretores e gêneros dos filmes. |
| **RF06** | Totalização de Público | O sistema deve gerar relatórios com a soma de público por sessão, por filme e por cinema. |
| **RF07** | Listagem de Cartaz | O sistema deve exibir os filmes que estão atualmente em exibição em cada unidade. |
| **RF08** | Autenticação | O sistema deve permitir que administradores e espectadores realizem login com e-mail e senha. |
| **RF09** | Recuperação de Senha | O sistema deve oferecer uma funcionalidade para recuperar a senha via e-mail. |
| **RF10** | Cadastro de Conta | O sistema deve permitir que novos espectadores criem uma conta para acessar funcionalidades exclusivas. |
