# 📊 Casos de Uso (UC)

Este documento apresenta a visão geral das interações entre os usuários e o sistema de gestão da Rede de Cinemas.

---

## 👥 Atores
- **Administrador / Funcionário:** Responsável pela gestão operacional, cadastros e geração de relatórios.
- **Espectador:** Usuário final que consome as informações de filmes e sessões.

## 📝 Lista de Funcionalidades
1.  **Manter Filmes:** CRUD completo de títulos.
2.  **Manter Unidades:** Gestão de infraestrutura física.
3.  **Gerenciar Sessões:** Alocação de horários e salas.
4.  **Registrar Público:** Controle de ocupação diária.
5.  **Gerar Relatórios:** Visão analítica de público.
6.  **Consultar Programação:** Busca por horários disponíveis.
7.  **Consultar Detalhes:** Informações técnicas de produções.
8.  **Realizar Login:** Controle de acesso seguro.
9.  **Cadastrar Conta:** Registro de novos espectadores.
